Sobre Mim

Sou desenvolvedor em transição de carreira, atualmente cursando Análise e Desenvolvimento de Sistemas na USCS. Com foco no ecossistema Java e no framework Spring Boot, especializo-me na criação de aplicações escaláveis, APIs RESTful e modelagem de bancos de dados relacionais com MySQL. Minha base técnica inclui certificações da FIAP e conhecimentos em infraestrutura de nuvem pela AWS, sempre priorizando as melhores práticas de Programação Orientada a Objetos e o desenvolvimento seguro.

Minha experiência profissional como Analista de Suporte de TI me proporcionou uma sólida capacidade de resolução de problemas críticos e manutenção de sistemas em ambientes de alta disponibilidade. Atualmente, busco minha primeira oportunidade como Estagiário Backend, onde possa aplicar meu domínio em metodologias ágeis (Scrum/Kanban) e versionamento de código para entregar soluções eficientes e de alto impacto.

Stacks Tecnológicas

Desenvolvimento Java

Tenho me dedicado ao Java (versões 11 e 17+) com foco em POO e escrita de código limpo. No dia a dia, utilizo Collections e Streams API para criar soluções eficientes, aplicando conceitos de programação funcional e lógica estruturada para garantir sistemas de fácil manutenção.

Springboot

Tenho desenvolvido APIs RESTful com Spring Boot, utilizando Spring Web, Spring Data JPA e banco H2 para persistência e testes. Gerencio meus projetos com Maven ou Gradle e prezo pela clareza das entregas, utilizando o Swagger para documentar as funcionalidades e facilitar a integração das soluções.

Banco Dados SQL

Trabalho com MySQL focado na organização e integridade dos dados por meio de modelagem relacional e estruturação de tabelas. No dia a dia, realizo operações de CRUD e consultas SQL eficientes para garantir que a persistência das informações suporte as regras de negócio com segurança.

Desenvolvimento Python

Utilizo Python como ferramenta de apoio para automação de scripts e análises de dados básicas. É uma linguagem que complementa meu perfil técnico, trazendo agilidade na criação de ferramentas auxiliares que otimizam tarefas do dia a dia e processos repetitivos.
