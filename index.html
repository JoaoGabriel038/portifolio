Sobre Mim

Sou desenvolvedor em transição de carreira, atualmente cursando Análise e Desenvolvimento de Sistemas na USCS. Com foco no ecossistema Java e no framework Spring Boot, especializo-me na criação de aplicações escaláveis, APIs RESTful e modelagem de bancos de dados relacionais com MySQL. Minha base técnica inclui certificações da FIAP e conhecimentos em infraestrutura de nuvem pela AWS, sempre priorizando as melhores práticas de Programação Orientada a Objetos e o desenvolvimento seguro.

Minha experiência profissional como Analista de Suporte de TI me proporcionou uma sólida capacidade de resolução de problemas críticos e manutenção de sistemas em ambientes de alta disponibilidade. Atualmente, busco minha primeira oportunidade como Estagiário Backend, onde possa aplicar meu domínio em metodologias ágeis (Scrum/Kanban) e versionamento de código para entregar soluções eficientes e de alto impacto.

Minhas habilidades

Desenvolvimento Java

Tenho me dedicado ao estudo aprofundado da Programação Orientada a Objetos (POO) e das funcionalidades modernas das versões 11 e 17+ do Java. No dia a dia, utilizo recursos como a Collections e Streams API para escrever um código mais limpo e eficiente. Atualmente, foco meus projetos no desenvolvimento de APIs RESTful e estou expandindo meus conhecimentos com o framework Spring Boot para construir soluções cada vez mais completas.

Desenvolvimento Python

Utilizo o Python principalmente como uma ferramenta de apoio para facilitar processos do dia a dia. Tenho focado no desenvolvimento de scripts de automação que otimizam tarefas repetitivas e na realização de análises de dados básicas. É uma linguagem que complementa meu perfil técnico, permitindo mais agilidade na criação de ferramentas auxiliares para os projetos em que atuo.

Banco Dados SQL

Trabalho com MySQL com foco na organização e integridade dos dados. Tenho experiência prática na modelagem relacional, estruturando tabelas e relacionamentos que suportem as regras de negócio das aplicações. No dia a dia, utilizo consultas SQL para manipulação de informações e operações de CRUD, garantindo que a persistência dos dados seja feita de forma eficiente e segura.

Desenvolvimento Web

No desenvolvimento de interfaces, foco em criar experiências que sejam fluidas e visualmente acessíveis para o usuário final. Tenho experiência prática na construção de interfaces responsivas com foco em performance e carregamento otimizado, garantindo que a aplicação funcione bem tanto no desktop quanto em dispositivos móveis.
